## Installation

### CocoaPods

Add the following line to your Podfile:

```ruby
pod 'ScrollableDatepicker'
```

Then run the following in the same directory as your Podfile:
```ruby
pod install
```

### Manual

Copy the folder `Sources/` to your project.
